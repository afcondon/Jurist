Petri nets → functorial dynamics

Each model below is a typed Petri net described once in PureScript — species and reactions, no differential equations. Handed to Julia's AlgebraicPetri, the mass-action dynamics is produced by a functor from the category of Petri nets to vector fields (not a hand translation); the derived ODEs are shown, and OrdinaryDiffEq integrates them. The browser draws a trajectory it never computed. There is no applied-category-theory stack in the JS/WASM world to hand a description to.
