Rigorous roots — proofs, not guesses

Each function is a typed NumExpr handed to Julia's IntervalRootFinding, which doesn't just find roots — it proves them: every root bracketed in a guaranteed enclosure, certified unique, with an exhaustive search so no root is missed — and "none" is itself a proof. A scipy/JS solver gives a number and a prayer.

The PureScript you write

let
  x = var "x"
  rootCases =
    [ { title: "x³ − 2x − 5", lo: -3.0, hi: 3.0
      , e: x * x * x - num 2.0 * x - num 5.0 }
    , { title: "sin x − x∕3  (three crossings)"
      , lo: -6.0, hi: 6.0
      , e: sinE x - divide x (num 3.0) }
    , { title: "x² + 1  — proven to have no real roots"
      , lo: -3.0, hi: 3.0
      , e: x * x + num 1.0 }
    , { title: "x⁴ − 10x² + 9  =  (x²−1)(x²−9)"
      , lo: -4.0, hi: 4.0
      , e: x * x * x * x - num 10.0 * (x * x) + num 9.0 }
    ]

-- for each case: proven enclosures, each certified
-- unique, none missed
rs <- provenRoots "x" c.lo c.hi c.e

Powers written as products so interval arithmetic stays valid over negative x. The expression crosses the seam once as a description; the proof comes back as data.
