Functorial data migration

A function-level call graph described once in PureScript, migrated functorially to its module dependency graph — a Δ migration along the in_mod functor, where every call f → g becomes a module edge in_mod(f) → in_mod(g). Not a hand-written fold; a migration functor with categorical guarantees. The result is a plain graph — exactly the shape Hylograph lays out and renders.
