Exact differentiation

You describe an expression once, as a typed value. Julia differentiates it symbolically and hands the derivative back — the chain rule, fractions and all — typeset, never written by hand. Descriptions across, descriptions back.

The PureScript you write

showcase :: NumExpr

  num 10.0 * pow (var "x") (num 2.0)
    - var "x" * var "y"
    + sinE (var "y")
    + logE (var "x")
