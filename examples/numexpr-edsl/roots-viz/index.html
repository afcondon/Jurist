Rigorous roots

Each function was described once in PureScript as a typed NumExpr and handed to Julia's IntervalRootFinding, which doesn't just find roots — it proves them: every root is bracketed in a guaranteed enclosure and certified unique, and the search is exhaustive, so no root is missed (and "none" is itself a proof). Validated numerics — the answer holds despite floating point. A `scipy`/JS solver gives a number and a prayer.
