Jurist demos

Small typed languages for mathematical things

Little languages for dynamical systems, symbolic expressions, optimization models, Petri nets, typed graphs — backed by the Julia scientific ecosystem as their production semantics.

Each is a handful of PureScript types: cheap to mint, one per domain, and a description that typechecks is a well-formed object of that domain.

Like any well-made eDSL, each admits more than one interpreter — and they are not equals:
