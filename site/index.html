You write the description.
Julia computes the un-computable.

Every demo below is the same move: a typed PureScript value — an ADT, a row-typed system, a typed graph — crosses the seam once, as a description, and Julia's runtime computes something structurally out of reach for JS, WASM, or scipy: symbolic derivatives, DAE solving, proven root enclosures, proven-optimal MILP, functorial dynamics and data migration. Each page counterposes the PureScript you write with what Julia computes from it.

  }
